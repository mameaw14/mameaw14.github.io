**\+ Likebox คืออะไร?**  
Likebox มีรูปร่างหน้าตาเป็นแบบนี้ค่ะ
![](http://upic.me/i/zt/likebox.png)
ฮาวทูนี้จะสอนทำกล่องแบบนี้ค่ะ 
อย่าเข้าใจผิดนะคะ ไม่ใช่แบบนี้
![](http://upic.me/i/qf/likebottom.png)
แบบนี้เรียกว่า Like Bottom ค่ะ

**\+ สิ่งที่ต้องมี**  
**Page on Facebook** เอาละสิ ทำยังไง อันนี้ขอให้ไปหาความรู้ักันนอกเอนทรี่นี้อีกทีนะคะ

**\+ howto**  
1. เข้าไปที่ [http://developers.facebook.com/docs.........ugins/like-box/](http://developers.facebook.com/docs/reference/plugins/like-box/)

2. จะเจอแบบภาพนี้ค่ะ
  ![](http://upic.me/i/5k/ytick.png)
  ![](http://upic.me/i/8e/2f5ex.png)
  Facebook Page URL = ใส่ URL ของเพจเราลงไปเล้ย  
  Width = ความกว้างของกล่อง  
  Color Scheme = สีของกล่อง มีสว่างกับมืด เลือกเอาค่ะ  
  Show Faces = เลือกว่าจะให้โชว์หน้าคนที่กดไลค์เราไหม?  
  Stream = โชว์เนื้อหาในเพจนั้นไหม?  
  Header = Find us on Facebook



3. จัดการกับตัวเลือกและขนาดให้เรียบร้อย

4. กดปุ่ม Get Code

5. ก๊อบโค้ดด้านล่างไปใช้ค่ะ (XFBML)
  ![](http://upic.me/i/3d/getcode.png)
  

**\+ วิธีใส่ Likebox ใน tumblr**  
1. เข้าไปที่ [http://www.tumblr.com/customize](http://www.tumblr.com/customize)
2. กด Info
3. ใส่โค้ดที่ได้มาลงในช่อง Description
4. เรียบร้อย
