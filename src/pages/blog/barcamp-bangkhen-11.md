<p><strong>#4 Nix: Declarative OS By awkwin</strong></p>



<p>พี่วินเปิด 2 sessions ที่เราอ่านหัวข้อแล้วไม่รู้ว่าคือไรทั้งคู่ แต่อันนี้มีคีย์เวิร์ดว่า Declarative OS ก็คงเป็น os ไรสักอย่างที่ declarative แหละ อยากรู้ว่าคืออะไร เลยมาฟัง ก็คือเป็น declarative os แหละ 55555 เขียน config declarative ว่าอยากให้มีไรบ้าง a b c มันก็จะลงให้เลย ทำให้ง่ายต่อการ manage จะลบก็ลบ เป็น functional compile แล้วต้องได้เหมือนเดิม เพราะฉะนั้นจะมี hash มาเทียบว่าเหมือนเดิมไหม แล้วก็ใช้ hash หา cache ด้วย ถ้า hash ตรงไม่ต้อง build ซ้ำ ไปโหลดตาม hash มาเลย </p>

<p><strong>#6 Geometry of LEGO By Neizod</strong></p>
