# mameaw14.xyz

personal website

## Prerequisites

Node LTS version (18)

## Setup Local
Install Playwright
```shell
npx playwright install
```

## Command

### Run

`npm run dev`

### Build

`npm run build`

### Test

#### Unit Tests
```shell
npm run test
```

#### E2E Tests
```shell
npm run build
npm run test:e2e
```
